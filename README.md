# Web Server using Docker

## CodeAlpha DevOps Internship - Task 4

### Project Description
This project demonstrates how to deploy a simple HTML website using Docker.

## Technologies Used
- Docker
- HTML
- Git
- GitHub

## Files
- Dockerfile
- index.html

## Build the Docker Image

```bash
docker build -t codealpha-webserver .
```

## Run the Docker Container

```bash
docker run -d -p 8081:80 codealpha-webserver
```

## Open the Website

http://localhost:8081
